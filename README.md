# 概要
これは、クソアプリアドベントカレンダー 2025 向けのクソアプリです。  
色選択ダイアログだけで、それ以上の機能はありません！  

# 対応プラットフォーム
Windows / macOS / iOS / Android

# 製作環境
Delphi 12.3
恐らく無料の Community Edition でも動きます。

# ビルド上の注意
OpenAI の API Key は APIKeys.inc という名前でプロジェクトと同じフォルダに保存してください。
APIKey はシングルクオートの付いた文字列としてください。

例：APIKeys.inc の中身
```text
'sk-honyarara-abcdefghijklmnopqrstuvwxyz'
```
